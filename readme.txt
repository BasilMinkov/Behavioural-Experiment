1. Кликаем по файлу install.bat правой кнопкой мыши, а затем кликаем "Запустить от имени администратора".
2. Откроется черное окно терминала. Это значит, что началась установка. Ждем, пока окно закроется.
Если появился файл run.bat, значит установка прошла успешно.
3. Переключаем язык на английский. Кликаем по файлу run.bat левой кнопкой мыши. Запускается эксперимент. 
4. Печатаем помер испытуемого. Начинается экспериментальная сессия. 
5. После экспериментальной сессии данные сипытуемого сохранятся в папку data.
